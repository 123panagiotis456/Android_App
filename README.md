# SupermarketManager

SupermarketManager is a sample Android application written in Kotlin. It showcases a small supermarket shopping experience with categories, product listings and a shopping cart.

## Features

- View product categories
- Browse products by category
- Search products
- Add items to a wishlist
- Manage a shopping cart with automatic total calculation
- View purchase history
- Uses Room for local data storage
- MVVM architecture with ViewModel and LiveData
- Navigation component for screen transitions

## Building

The project is built using Gradle. You can open it with **Android Studio** or build from the command line:

```bash
./gradlew assembleDebug
```

Minimum SDK version is 24 and the project targets SDK 34. Java 17 is used.

## Running

After building, install the generated APK on an emulator or device running Android 7.0 or higher. When launched you can browse categories, add products to your cart and track past purchases.
